# Remotion 실습 프로젝트

안녕하세요! 바이브코더 루피입니다.

원데이클래스에서 Remotion을 배우면서 실습한 코드들을 정리해서 올려봅니다.
React로 영상을 만들 수 있다는 게 정말 신기하더라고요. 코드를 작성하면 그게 그대로 영상이 됩니다.

코딩이 처음이신 분들도 부담 없이 따라해보실 수 있도록 최대한 쉽게 정리해봤어요.
모르는 부분이 있으시면 언제든 이슈 남겨주세요. 같이 해결해봐요!

## 어떤 걸 만들었나요?

### 1. 포트폴리오 인트로 영상
터미널 해커 감성의 자기소개 영상이에요.
- 타이핑 애니메이션이 한 글자씩 나타나는 효과
- 글리치(화면 깨짐) 효과와 스캔라인
- 프로그레스 바가 차오르는 연출 (`[████████░░░░] 95%`)
- 장면이 바뀔 때 플래시 전환 효과

### 2. SRT 기반 유튜브 영상 자동화
자막 파일(SRT)만 넣으면 영상이 자동으로 만들어지는 시스템이에요.
- SRT 파일을 읽어서 자동으로 장면을 나눠줍니다 (자막 3개가 1장면)
- 지금 읽히는 단어가 하이라이트 되는 자막 (틱톡에서 많이 보이는 그 느낌이에요)
- 화면 왼쪽에는 장면에 맞는 모션그래픽, 오른쪽에는 자막이 나옵니다
- 장면마다 컬러 테마가 자동으로 바뀌어서 지루하지 않아요

## 실행 방법

> 아래 명령어는 터미널(명령 프롬프트)에서 입력하시면 됩니다.
> VS Code를 사용하시는 분은 상단 메뉴에서 `터미널 > 새 터미널`을 열어주세요.

```bash
# 1. 필요한 패키지 설치 (처음 한 번만 하면 돼요)
npm install

# 2. 미리보기 실행 (브라우저에서 바로 확인할 수 있어요!)
npm run dev

# 3. 완성된 영상을 MP4 파일로 내보내기
npx remotion render SrtVideo
```

## 프로젝트 구조

> 각 파일이 어떤 역할을 하는지 간단하게 적어뒀어요.

```
src/
├── Composition.tsx    # 영상의 전체 구성을 담당하는 메인 파일
├── CaptionText.tsx    # 단어별로 하이라이트 되는 자막 컴포넌트
├── SceneVisuals.tsx   # 각 장면에 들어가는 모션그래픽 애니메이션
├── SubtitleScene.tsx  # 장면 전환 시 배경이 바뀌는 효과
├── srtParser.ts       # SRT 자막 파일을 읽어서 데이터로 변환하는 도구
├── Root.tsx           # 영상의 시작점 (해상도, 길이 등을 설정)
└── index.ts
sample.srt             # 테스트용 예시 자막 파일 (약 1분 분량)
```

## 장면별 모션그래픽 구성

| 장면 | 왼쪽 비주얼 | 설명 |
|------|-----------|------|
| INTRO | 터미널 화면 | `$ vibe create my-app` 명령어가 타이핑되는 연출 |
| AI | 뉴런 네트워크 | 인공지능 노드들이 서로 연결되며 회전 |
| TOOLS | 도구 아이콘 | Cursor, Claude, Copilot이 차례로 등장 |
| INSIGHT | 장벽 붕괴 | 블록들이 부서지면서 OPEN 텍스트가 나타남 |
| POWER | 숫자 카운터 | 10x가 탄력 있게 튀어오르는 애니메이션 |
| OUTRO | 프롬프트 창 | 입력창에 글자가 타이핑되는 마무리 연출 |

## 사용 기술

## 내 컴퓨터에서 직접 실행해보기

> Git이나 Node.js가 처음이셔도 괜찮아요. 아래 순서대로만 따라하시면 됩니다.

### 사전 준비

아래 두 가지가 설치되어 있어야 해요. 아직 없으시다면 링크에서 설치해주세요.
- [Node.js](https://nodejs.org/) (LTS 버전 추천)
- [Git](https://git-scm.com/downloads)

### 1단계: 프로젝트 복사하기

터미널을 열고 아래 명령어를 입력해주세요. 이 프로젝트가 내 컴퓨터로 복사됩니다.

```bash
git clone https://github.com/vibecoder-loopy/portfolio-intro.git
cd portfolio-intro
npm install
```

### 2단계: 미리보기로 확인하기

```bash
npm run dev
```

브라우저가 자동으로 열리면서 영상을 바로 확인할 수 있어요.
타임라인을 드래그하면서 각 장면을 살펴보세요!

### 3단계: 내 맘대로 수정해보기

`src/Composition.tsx` 파일을 열어서 SRT 텍스트를 바꿔보세요.
저장하면 브라우저에서 바로 반영됩니다. 이게 Remotion의 매력이에요!

## Claude Code로 수정하는 방법

> 코드를 직접 수정하기 어려우시다면, Claude Code에게 부탁하면 됩니다.
> 자연어로 말하면 알아서 코드를 수정해줘요.

### Claude Code 설치

```bash
npm install -g @anthropic-ai/claude-code
```

### 프로젝트 열기

```bash
cd portfolio-intro
claude
```

### 이렇게 말해보세요

Claude Code가 실행되면, 아래처럼 자연어로 요청하시면 돼요.

```
"sample.srt 파일 내용을 내 유튜브 대본으로 바꿔줘"

"자막 글씨 크기 좀 더 키워줘"

"배경색을 검정 대신 진한 남색으로 바꿔줘"

"장면 전환할 때 페이드 효과 넣어줘"

"새로운 SRT 파일로 영상 만들어줘"
```

코딩을 몰라도 대화하듯이 수정할 수 있습니다.
원하는 게 있으면 편하게 말해보세요. Claude Code가 알아서 해줄 거예요!
